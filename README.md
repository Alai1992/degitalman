# kmycwar - 抗美援朝老兵数字人网页应用

这是一个基于抗美援朝历史主题的数字人教学与互动探究网页应用。

## 功能模块

1. **数字人授课** - 聆听老兵讲述峥嵘岁月，观看历史视频
2. **互动探究** - 与老兵数字人对话，探索历史细节
3. **学习支持** - 丰富的历史文献、影像、音频资源

## 技术栈

- 前端：Vite + TypeScript + Tailwind CSS
- 后端：Express + TypeScript
- AI：大语言模型流式对话

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 生产构建
pnpm run build
pnpm run start
```

## 项目结构

```
├── public/           # 前端页面和静态资源
├── src/             # 前端源码
├── server/          # 后端API
└── index.html       # 主页入口
```
