# 抗美援朝老兵数字人项目 - 开发记录

## 项目概述

本项目是一个抗美援朝老兵数字人展示网站，包含数字人授课、互动探究、知识宝库三大模块。

**技术栈**：Vite + TypeScript + Express + Tailwind CSS
**访问域名**：${COZE_PROJECT_DOMAIN_DEFAULT}

---

## 一、主页 (index.html / src/index.css)

### 1.1 页面结构
- 战火背景视频
- 主标题：云寻老兵烽火迹
- 纪念74周年徽章
- 三个功能模块按钮

## 二、数字人授课页面 (digital-human.html)

### 2.1 功能说明
- 展示老兵数字人形象
- 点击开始按钮启动数字人讲解
- 讲解完成后显示历史简介内容

### 3.1 功能说明

- 音频播放功能

## 五、子页面返回功能

### 5.1 问题修复
子页面返回知识宝库时，自动跳转到之前选择的tab（栏位）。

**实现方式**：
- URL参数传递：?from=tabName
- 支持的tab：maps、documents、audio

## 七、API接口

### 7.1 TTS接口 (server/routes/tts.ts)
- **端点**：POST /api/tts
- **功能**：文字转语音
- **音色配置**：
  - 音色：如雅逸尘 (zh_male_ruyayichen_saturn_bigtts)
  - 语速：-0.15（较慢）
  - 采样率：24000Hz
