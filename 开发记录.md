# 抗美援朝老兵数字人项目 - 完整开发记录

## 项目概述

本项目是一个抗美援朝老兵数字人展示网站"云寻老兵烽火迹"，旨在纪念抗美援朝战争74周年。

**项目名称**：抗美援朝老兵数字人
**技术栈**：Vite + TypeScript + Express + Tailwind CSS
**访问域名**：${COZE_PROJECT_DOMAIN_DEFAULT}

---

**实现方式**：
```javascript
// 统一管理音频播放
const audioElements = {
  song: document.getElementById('audioPlayer'),
  heroSong: document.getElementById('heroAudioPlayer')
};

function toggleAudio(type) {
  // 停止其他音频
  if (type !== 'song') stopAudio('song');
  if (type !== 'hero') stopAudio('hero');
  // 播放当前音频
  audioElements[type].play();
}
```

### 3.5 问题修复

#### 问题1：子页面返回时tab状态丢失

**用户请求**：返回知识宝库时，展示的栏位为之前选择的栏位

**问题分析**：从子页面返回时，不知道用户之前在哪个tab

**解决方案**：
1. 修改所有子页面跳转链接，添加参数
```javascript
// 从知识宝库进入子页面时
location.href = 'map-korea-terrain.html?from=maps'

// 子页面返回时
goBack() {
  const params = new URLSearchParams(window.location.search);
  const from = params.get('from') || 'audio';
  location.href = `learning-support.html?tab=${from}`;
}
```

2. 修改知识宝库页面
```javascript
// 检查URL参数
const params = new URLSearchParams(window.location.search);
const tab = params.get('tab');
if (tab) switchTab(tab);
```

**修改文件**：
- `/workspace/projects/public/learning-support.html`
- `/workspace/projects/public/map-korea-terrain.html`
- `/workspace/projects/public/map-five-campaigns.html`
- `/workspace/projects/public/war-diary.html`

#### 问题2：音频播放互斥

**用户请求**：播放音乐时，如有另一首在播放，需要停止另一首

**解决方案**：
```javascript
function toggleAudio(type) {
  // 先停止所有音频
  stopAudio('song');
  stopAudio('hero');
  
  // 根据类型播放
  if (type === 'song') {
    audioPlayer.play();
    isPlaying.song = true;
    updateButton('song', '停止', true);
  } else {
    heroAudioPlayer.play();
    isPlaying.hero = true;
    updateButton('hero', '停止', true);
  }
}
```

**修改文件**：`/workspace/projects/public/learning-support.html`

```javascript
let currentPlaying = null;

  if (currentPlaying && currentPlaying !== type) {
    stopAudio(currentPlaying);
  }
  // ... 播放当前音频
  currentPlaying = type;
}
```
