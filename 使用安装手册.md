# 抗美援朝老兵数字人项目 - 使用安装手册

## 一、项目简介

本项目是一个抗美援朝老兵数字人纪念网站，包含三大核心模块：

| 模块 | 功能 |
|------|------|
| 数字人授课 | 老兵数字人讲解历史 |
| 互动探究 | AI对话老兵、珍贵影像 |
| 知识宝库 | 音频资源、历史地图、文献档案 |

## 二、环境要求

- **Node.js**: v18 或更高版本
- **包管理器**: pnpm（必须使用，禁止使用 npm 或 yarn）
- **端口**: 5000（服务监听端口）

## 四、安装步骤

### 1. 安装依赖

```bash
cd /workspace/projects/
pnpm install
```

### 2. 启动开发环境

```bash
pnpm dev
```

服务将在 **http://localhost:5000** 启动

### 3. 构建生产版本

```bash
pnpm build
```

### 4. 启动生产环境

```bash
pnpm start
```

## 七、功能说明

### 7.1 主页
- 展示项目主题和三大模块入口
- 点击模块按钮进入对应功能页面

### 7.2 数字人授课
- 老兵数字人自动讲解抗美援朝历史

### 7.3 互动探究
- **老兵对话**：与92岁老战士杨德清AI对话
- **珍贵影像**：观看历史影像资料
- AI对话使用苍老音色（邵云/如雅逸尘）

### 7.4 知识宝库
- **音频资源**：播放《志愿军战歌》《英雄赞歌》
- **历史地图**：朝鲜半岛地形图、五次战役作战图
- **文献档案**：战地日记选编

## 八、API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/chat` | POST | AI对话接口（SSE流式输出） |
| `/api/chat/history` | GET | 获取对话历史 |
| `/api/chat/history` | DELETE | 清除对话历史 |
| `/api/tts` | POST | 语音合成接口 |

## 九、注意事项

1. **包管理器**：必须使用 `pnpm`，不要使用 npm 或 yarn
2. **端口**：服务必须运行在 5000 端口
3. **静态资源**：音频、图片等资源放在 `public/` 目录下
4. **对话人设**：AI角色为92岁山东老兵杨德清，回复需用纯中文

## 十、常见问题

### Q: 安装依赖失败
```bash
# 清除缓存后重试
pnpm store prune
pnpm install
```

### Q: 端口被占用
```bash
# 查找占用端口的进程
lsof -i:5000
# 或使用其他端口启动（需修改配置）
```

### Q: AI对话无响应
检查 `/api/chat` 接口是否正常，确认 LLM 服务配置正确。
